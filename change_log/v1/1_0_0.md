# HARMONIS BACK OFFICE PORTAL V1.0.0

## HRMS Module
- **Manajemen Kehadiran (Attendance)**
    - Pencatatan kehadiran karyawan (check-in/check-out).
    - Penyimpanan evidence & data kehadiran dari integrasi mobile.
- **Timesheet / Daily Task**
    - Pencatatan pekerjaan harian.
    - Review/approval oleh atasan.
- **Leave & Overtime Request**
    - Pengajuan cuti dan lembur.
    - Mekanisme approval multi-level.
- **Payroll Processing**
    - Perhitungan gaji berdasarkan komponen gaji, kehadiran, dan lembur.
    - Approval workflow payroll.
    - Locking periode setelah disetujui.
- **Salary Slip**
    - Penerbitan slip gaji digital.

## Transaction Module
- **Invoice Management**
    - Pembuatan dan penerbitan invoice.
    - Workflow approval berdasarkan level kewenangan.
    - Integrasi Customer & Service Master Data.

## Master Data
- Customer Master Data
- Service Master Data
- Employee Master Data
- User Master Data
- Asset Master Data
- Salary Component Master
- Organizations Master  

## Inventory Module
- Asset Movement Summary
- Asset Movement
- Asset Transaction  

## BAST Projects Module
- Import atau input data BAST Project.
- Penugasan pekerjaan kepada teknisi.
- Penyajian evidence dari lapangan (koordinat & foto).
- Perhitungan progres berdasarkan evidence.
- Monitoring & laporan real-time.

## Non-Functional
- Activity Log untuk seluruh aktivitas Web App.
